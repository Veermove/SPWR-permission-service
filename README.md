# pwrspotted

Microservice skeleton.

<br />




## how?

* clone repo and install dependencies
    ```bash
    $ git clone ...
    $ cd pwrspotted
    $ npm i
    ```

* build and run
    ```bash
    $ npm run build
    $ npm start
    ```

* (optional) want to tinker yourself?
    ```bash
    $ npm run dev
    ```
